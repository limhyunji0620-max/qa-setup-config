# 🚀 QA Setup Launcher (통합 설치 프로그램)

PowerShell을 기반으로 제작된 **QA 업무 필수 소프트웨어 통합 설치 도구**입니다. 
복잡한 설정 없이 한 번의 클릭으로 테스트 환경에 필요한 유틸리티와 전용 도구들을 자동 설치합니다.

---

## ✨ 주요 기능

* **자동화 설치**: Chrome, 반디집, 픽픽 등 필수 프로그램을 Silent 모드로 자동 설치합니다.
* **MS Office 2024 배포**: ODT(Office Deployment Tool)를 통해 XML 구성을 자동 생성하고 정식 배포 버전으로 설치합니다.
* **전용 도구 고정**: 'RawData 자동 캡처 프로그램'을 내 문서 폴더에 고정하고 바탕화면 바로가기를 생성합니다.
* **실시간 상태 확인**: 하단 상태바를 통해 현재 작업 중인 프로세스를 확인할 수 있습니다.
* **설치 제어**: 전체 선택/해제 기능 및 설치 중 안전 중단 기능을 제공합니다.

---

| :--- | :---: | :---: | :--- |

## 🛠 사용 방법

1.  **관리자 권한으로 실행**
    * 시스템 환경 설정을 위해 PowerShell을 **관리자 권한**으로 실행해 주세요.
2.  **스크립트 실행 허용** (필요 시)
    * 스크립트 실행이 제한된 경우, PowerShell 창에 아래 명령어를 입력하세요:
        ```powershell
        Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
        ```
3.  **런처 조작**
    * `QA_Setup_Launcher.ps1` 파일을 실행합니다.
    * 설치할 프로그램을 체크한 후 **[프로그램 설치 시작]** 버튼을 클릭합니다.
    * 설치가 진행되는 동안 상태바의 메시지를 확인해 주세요.

---

## ⚠️ 주의 사항

* **네트워크 환경**: 설치 파일을 외부 서버에서 실시간으로 다운로드하므로 인터넷 연결이 반드시 필요합니다.
* **설치 중단**: [설치 취소] 클릭 시 현재 진행 중인 다운로드는 즉시 중단되나, 이미 실행된 설치 창(Installer)은 수동으로 닫아주어야 안전합니다.
* **보안 프로그램**: 일부 백신 프로그램에서 실시간 스크립트 실행을 차단할 수 있으니 확인이 필요합니다.
